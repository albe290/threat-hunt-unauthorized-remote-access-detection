# Splunk Query Workflow

These queries demonstrate the full threat hunting lifecycle:

1. Discovery – Identify presence of suspicious activity  
2. Scoping – Group activity by host and user  
3. Deep Analysis – Parse telemetry and extract fields  
4. Validation – Review raw events for accuracy  
5. Aggregation – Summarize behavior across time  
6. Detection Development – Identify patterns for alerting  
7. Outlier Analysis – Separate benign vs suspicious activity  

All queries have been sanitized to remove environment-specific identifiers while preserving detection logic and methodology.
